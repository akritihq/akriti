Certified topological data analysis with per-prediction guarantees.

Library in development. First public release expected late 2026. Underlying papers (PLACE, PALACE, CASTLE) under JMLR submission.

The four products

akriti.place

Closed-form persistence-landmark embedding with explicit bi-Lipschitz distortion bound. Tuning-free.

akriti.palace

Adaptive-landmark RKHS lift with closed-form filtration selection and per-prediction kernel-margin certificates.

akriti.castle

Applied-statistics toolkit: certified two-sample test, sample-size calculator, robustness certificate.

akriti.core

Shared primitives: bottleneck distance, top-N filter, certificate radii, GPU-batched embedding ops.

Why Akriti

  • Per-prediction certificates — not just Lipschitz upper bounds.
  • Closed-form descriptor selectors. No grid search.
  • GPU-batched embeddings on CPU, CUDA, and Apple MPS.
  • sklearn estimators and PyTorch Geometric layers.
  • Apache-2.0 — permissive, industry-friendly.
